TITULO: Assinatura Avançada

---------------------------------------------------

DESCRICAO DO PROJETO

Assinatura Avançada é um projeto Full-Stack desenvolvido para simular e gerenciar um sistema de assinatura digital robusto e avançado. O objetivo é fornecer uma solução completa, com lógica de back-end segura e interfaces de usuário separadas para a web e para dispositivos móveis.

Este projeto é uma demonstração de habilidades em arquitetura de software, desenvolvimento de API RESTful e construção de interfaces de usuário com foco em tipagem e escalabilidade.

ARQUITETURA DO PROJETO

O repositório é organizado em módulos distintos:
* API de Assinatura (Caminho: digital-signer-api/src): Lógica de validação, criptografia, autenticação da assinatura digital e persistência de dados.
* Aplicação Mobile (Caminho: new-stable-mobile): Interface do usuário para dispositivos móveis, permitindo a visualização e submissão de assinaturas.
* Web Client (Caminho: Arquivos raiz - index.html, script.js): Interface básica para demonstração em navegador.

---------------------------------------------------

FUNCIONALIDADES CHAVE

* Gerenciamento de Assinaturas: Criação e validação de documentos assinados digitalmente.
* API RESTful: Endpoints bem definidos para comunicação entre o cliente e a lógica de assinatura.
* Suporte Multiplataforma: Solução integrada para acesso via web e aplicativo mobile.
* Tipagem Forte: Uso de TypeScript para código mais seguro e de fácil manutenção.
